About

Welcome to MemeRank, where dreams become memes, or something like that. Created by 4 Northeatern Second Year CS students during the Fall Hackathon Husky Hacks to commemorate their true memer friend Ethan March. Memes uploaded here are ranked based on a scale created by Ethan March, who the website it dedicated to, and the 5/7 top rank is a meme in itself. (Link for those of you who still need to be educated) An actual use of this would be to check and see if your meme is worthy before you post it so your friends think you are actually funny.

How it Works

First, we scrape data from subreddits like /r/me_irl and we store the URL and the upvote count. From there we use a script written in Java to pass the images through the Clarifai API which returns the top 5 tags that are associated with that image (i.e. for a red truck it'll be 'red', 'car', 'big', etc). As each image has its tags generated, using the same Java code, we pass save all of the previous attributes to a MySQL database. When a user inputs an image to the site, it is run through the API using a PHP script and its tags are then checked with the tags of other images in the database. For each tag that the image has in common with other images in the database, we sum the upvote count of the matched image. We then do some more math magic to rank the images based on our scale, but this description is already long enough. The site is hosted on a DigitalOcean droplet with a LAMP stack and uses Jeet as the framework for this amazing site.
